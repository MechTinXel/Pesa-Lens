## 📥 Installation Instructions

### For Your Friend (or Any User)

1. **Visit**: https://github.com/MechTinXel/Pesa-Lens/releases
2. **Download**: Click on the latest release
3. **Get APK**: Download `app-debug.apk` (usually smaller, same functionality)
4. **Enable Unknown Sources**:
   - Settings → Apps → Special app access → Install unknown apps
   - Toggle your browser/file manager ON
5. **Install**: Tap the APK and select "Install"
6. **Grant Permissions**: 
   - SMS Read permission
   - Biometric permission
7. **Use**: Open the app and start tracking!
